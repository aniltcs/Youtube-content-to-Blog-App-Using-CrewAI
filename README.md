# Youtube-content-to-Blog-App-Using-CrewAI

Created an app that have 2 agents:

One agent is Researcher who reads content from youtube video using the Crew AI tool: YoutubeVideoSearchTool

Another agent is Blog writer who writes blogs using content from Researcher.

We have used Crew AI agents,Open AI model in crew AI and "Process.sequential" Crew AI process so output from researcher will be forwarded to blog writer agent.

App Url: https://youtube-content-to-blog-app-using-crewai.streamlit.app/
